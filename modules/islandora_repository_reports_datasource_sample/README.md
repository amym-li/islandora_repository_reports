# Sample Data Source Plugin for Islandora Repository Reports

Sample data source plugin for the Islandora Repository Reports module.

## Requirements

* [Islandora Repository Reports](https://github.com/mjordan/islandora_repository_reports)

## Installation

Enable the module either under the "Admin > Extend" menu or by running `drush en -y islandora_repository_reports_datasource_sample`.
